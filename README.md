# Movie Watchlist

A personal movie watchlist application built with Vite, JavaScript, CSS, and HTML.

## About

Movie Watchlist is a web application that allows users to maintain a personalized list of movies they want to watch. Users can add movies, mark them as watched, and organize their watchlist with ease.

## Tech Stack

- **Frontend:** JavaScript (56.5%), CSS (26.8%), HTML (16.7%)
- **Build Tool:** Vite
- **Language:** JavaScript, CSS, HTML

## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn package manager

### Installation

Install the dependencies:

```bash
npm install
# or
yarn install
```

### Development

Run the development server:

```bash
npm start
# or
yarn start
```

The application will be available at `http://localhost:5173` (or the port shown in your terminal).

## Features

- Add movies to your watchlist
- Mark movies as watched
- Remove movies from the list
- Search for movies
- Persistent storage using browser local storage
- Responsive and user-friendly interface
- Fast and interactive experience

## How to Use

1. Start the application
2. Search for movies using the search bar
3. Add movies to your watchlist
4. Mark movies as watched when you complete them
5. Remove movies you no longer want to watch
6. Your watchlist is automatically saved

## Build for Production

```bash
npm run build
# or
yarn build
```

## Data Storage

The application uses the browser's `localStorage` to persist your watchlist, so your data is saved even after closing the browser.
